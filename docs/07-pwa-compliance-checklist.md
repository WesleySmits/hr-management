# 07 — PWA Compliance Checklist

## Installability
- [ ] Valid web app manifest
- [ ] Required icons and maskable icons
- [ ] Start URL and display mode defined

## Offline & Resilience
- [ ] Service worker strategy documented
- [ ] Offline fallback behavior defined
- [ ] Retry/sync strategy defined for queued actions

## UX Requirements
- [ ] App shell loads fast on mobile
- [ ] Offline messaging clear and actionable
- [ ] Reconnection and sync states visible

## Security
- [ ] HTTPS required in production
- [ ] Sensitive data not cached unsafely

## Compliance Definition of Done
PWA criteria are mapped to implementation tasks with acceptance criteria and test notes.

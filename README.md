# HR Management — Context Pack (Planning Only)

## Status
Planning phase only. **No code implementation is allowed** until explicit permission is given.

## Goal
Build an AI-first HR platform with:
- strict TypeScript (when implementation starts)
- secure APIs where relevant
- full PWA compliance
- mobile-first, high-performance UX
- pluggable storage via Strategy Pattern

## Current Storage Direction
Start with a **native browser database** for speed of delivery in early iterations, while designing the storage layer to support migration to Postgres later without feature rewrites.
